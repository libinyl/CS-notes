## json



```json
//空对象
{}
//对象有两个元素,每个元素都是一个name-value 对
{"firstName": "Brett", "lastName": "McLaughlin"}  

// 一个对象,内有一个元素,这个元素的值是一个数组,数组中的成员是两个对象.
{ 
    "people":[ 
        {
            "firstName": "Brett",            
            "lastName":"McLaughlin"        
        },      
        {        
            "firstName":"Jason",
            "lastName":"Hunter"
        }
    ]
}
// 数组的每个成员可以是对象
// 数组的每个成员不可是键值对.
// 对象的成员一定是 n 个键值对.
// 值可以是对象,不可以是数组.

```
